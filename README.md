# 💳 RUID: O Fim da Fila no RU

**Acesso ultrarrápido e inteligência de dados para o Restaurante Universitário da UFAL.**

Projeto desenvolvido como solução para otimizar o fluxo de alunos no Restaurante Universitário (RU) para a disciplina de Gestão de Projetos referente à residência em Tecnologias RFID do laboratório EASY da UFAL. O RUID substitui a lenta digitação manual de matrículas por validação *contactless* via **RFID de Ultra Alta Frequência (UHF)**, combinando hardware de baixo custo com **Ciência de Dados** para prever o tempo de espera nas filas.

---

## ✨ Principais Funcionalidades

*   📱 **Integração "Minha UFAL":** Fluxo nativo para solicitar a tag RFID e recarregar saldo via PIX de forma expressa.
*   ⚡ **Validação Contactless (Totem):** Pagamento processado em milissegundos com feedback visual e sonoro imediato.
*   📊 **Inteligência de Fila (Data Science):** Geração de estimativas de espera em tempo real baseadas na vazão de pagamentos do totem.
*   📈 **Gestão Baseada em Dados:** Dashboards de uso diário e semanal para auxiliar a administração do RU no preparo de refeições.
